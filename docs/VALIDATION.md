## 方法說明

- **切割點範圍**：最早可讓 ≥40 檔股票有 ≥60 根K棒的日期，至 最新日期 − 30 天
- **前進horizon**：固定30個日曆日（修正了 backtest_holdout.py 的不一致horizon問題）
  - exit_price = 切割日後、30個日曆日內的最後一個收盤價
- **零前瞻保證**：計算指標前先截斷K棒至 <= 切割日；score_symbol 以 now=cutoff 呼叫
- **零捏造**：所有報酬 = exit_close / entry_close − 1，來自 SQLite 真實列
